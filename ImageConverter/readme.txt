★ 用途
画像ファイル(GIF/JPEGファイル)を読み込んで、スクウェアアートやTABLEアートに変換します。

★ 動作環境
JRE Standard Edition Version 1.4.2 以降のバージョンをインストール済みであれば動くと思います。

★ 起動方法
ImageConverter.jarをダブルクリックして下さい。
もしくはDOSプロンプトを立ち上げてImageConverter.jarを格納しているディレクトリまで移り、以下のように入力して下さい。
java -jar ImageConverter.jar
しばらくすると、画面が表示されます。

★ 使用方法
1. 画面中央のテキストエリアに画像ファイル(GIFもしくはJPEGファイル)をドラッグ&ドロップします。
   もしくは、メニューバーの[ファイル]-[開く]から画像ファイルを選択します。
   そうすると、画面上部に読み込んだ画像ファイルの情報が表示されます。
2. 次に変換する方式として「■アート」か「TABLEアート」のどちらかを選択します。
   このとき、変換後のHTMLソースのサイズが「サイズ（変換後）」欄に表示されます。
3.変換ボタンを押して変換処理を開始します。
4.変換処理が完了すると、テキストエリアに変換後のHTMLソースが表示されます。
5.メニューバーの[ファイル]-[HTMLファイルに保存]から変換後のHTMLソースをHTMLファイルに保存できます。

★ 注意事項
変換後のサイズが大きすぎるとブラウザでの表示処理がかなり重くなります。
場合によってはブラウザが落ちてしまうかもしれないので、気を付けましょう。
マシンのパワーにもよりますが、変換後のHTMLファイルのサイズが1MBをこえない程度にしましょう。
