★ 用途
携帯でよく使われる顔文字を2ch顔文字に変換します。
携帯顔文字を見ると虫酸がはしるような方にはお薦めのツールです。（´∀｀ ）

★ 動作環境
JRE Standard Edition Version 1.4.0 以降のバージョンをインストール済みであれば動きます。
Javaアプリケーションなので、JREがインストールされていればWindowsだろうがLinuxだろうが動作するはずです。

★ 起動方法
MonaConverter.jarをダブルクリックして下さい。
もしくはDOSプロンプト（ターミナル）を立ち上げてMonaConverter.jarを格納しているディレクトリまで移り、以下のように入力して下さい。
java -jar MonaConverter.jar
しばらくすると、画面にフレームが表示されます。（起動までに少し時間がかかるかもしれません）

★ 使用方法
テキストエリアに変換したい文章を入力し、[変換開始]ボタンをクリックすると変換後の文章がテキストエリアに表示されます。
テキストファイルを読み込んで変換することもできます。（ファイルのドラッグ＆ドロップに対応）
また、自分で変換データを追加することもできます。

★ 注意
sys_cnv_def.db（システム定義の顔文字変換データファイル）の文字コードはShift-JIS（改行コードはCR+LF）になっています。
UNIX系のOS上で実行する場合には、文字コードをEUCに変換してください。
